# 범용 원칙 (모든 프로젝트 공통)

## 배포 안전 원칙
1. 환경별 설정값 확인
2. 빌드 성공 확인
3. 빌드 산출물 포함

- 커밋 전 빌드 확인
- 미커밋 파일 없는 상태로 세션 종료
