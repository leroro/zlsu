# Cost Advisor 에이전트 메모리

## ZLSU 비용 구조 (현재)

| 항목 | 현재 | 비용 | 향후 리스크 |
|------|------|------|-----------|
| 호스팅 | GitHub Pages | 무료 | 트래픽 제한 있으나 동호회 규모에서 문제 없음 |
| 데이터 | localStorage | 무료 | DB 마이그레이션 시 비용 발생 |
| 인증 | 자체 구현 | 무료 | SNS 로그인 시 외부 서비스 비용 |
| AI 에이전트 | Claude API | 세션당 과금 | 모델 선택이 비용에 직결 |

## 에이전트 모델 배분 근거

| 에이전트 | 모델 | 근거 |
|---------|------|------|
| product-owner | Opus (inherit) | 정책 판단, 맥락 이해 → 고도의 추론 필요 |
| spec-analyst | Opus (inherit) | 복잡한 요구사항 분석, 데이터 설계 |
| ux-writer | Opus (inherit) | 섬세한 언어 감각, 인지부하 판단 |
| ui-developer | Opus (inherit) | UI 구현 판단, 패턴 매칭 |
| data-developer | Opus (inherit) | 타입 정확성, 데이터 정합성 |
| code-reviewer | Sonnet | 패턴 매칭 위주, 창의성 불필요 |
| qa-tester | Sonnet | 검증 실행 위주, 시나리오 따라감 |
| devops | Haiku | 정해진 절차 실행, 판단 최소 |
| cost-advisor | Sonnet | 분석적이나 고도의 창의성 불필요 |

## 알려진 기술 부채

| 항목 | 심각도 | 마이그레이션 비용 |
|------|--------|----------------|
| localStorage → DB | 중 | 높음 (전체 API 레이어 교체) |
| SNS 로그인 미구현 | 낮 | 중간 (OAuth 연동) |
| 테스트 코드 없음 | 중 | 중간 (점진적 추가 가능) |
| 자체 인증 → 표준 인증 | 낮 | 높음 (세션/토큰 구조 변경) |
