# Product Owner 에이전트 메모리

## ZLSU 서비스 철학

### 핵심 가치
- **접근성**: 30~50대 비기술 사용자가 주 타겟. 복잡한 기능보다 쉬운 사용이 우선
- **최소 운영 부담**: 관리자(총무) 1명이 대부분 운영. 관리 기능은 간결해야 함
- **카카오톡 보완**: 메인 소통은 카카오톡, 이 시스템은 공식 절차(가입, 회비 등)만 담당

### 기능 추가 판단 기준
```
1. 이 기능 없이 카카오톡으로 해결 가능한가?
   → YES: 만들지 않는다 (카톡이 더 편함)
   → NO: 이 시스템에서만 가능한 공식 절차인가?
     → YES: 구현
     → NO: 보류
```

## 정책 레퍼런스

- 회칙: `public/rules/v2.0.md`
- 개인정보: 이름, 연락처, 이메일만 수집 (모임 운영 목적)
- 회비: 월 5만원, 가입비 별도

## 이해관계자 관점

| 이해관계자 | 우선시하는 것 | 주의할 점 |
|-----------|-------------|----------|
| 총무 (임미선) | 운영 효율, 한눈에 파악 | 새 기능 = 관리 부담 증가 |
| 코치 (한선우) | 수업 운영, 레벨 관리 | 출결/레벨 외 기능에 관심 낮음 |
| 일반 회원 | 정보 확인, 간단한 조작 | 기술 문해력 낮음 (30~50대) |
| 신규 가입자 | 절차 이해, 진입 장벽 낮음 | 첫인상이 가입 전환율 결정 |
