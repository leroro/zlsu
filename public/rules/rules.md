# 즐수팀 운영규정

## 모임 안내

- 매주 토요일 8:00 ~ 8:50 수영 연습
- 선우 코치님 지도하에 경기 연습 위주 진행
- 수영 후 메가커피 티타임 (자율 참석)

## 회비 안내

- **월 회비**: 2만원 (매월 1일 납부)
- **계좌**: 카카오뱅크 79421007218 (임미선)
- 이미 납부한 회비는 환불되지 않습니다
- 탈퇴 또는 휴면 전환 시에도 기 납부 회비는 반환되지 않습니다

## 시간 준수 규정

- **8시 정각 도착** (수영장 시계 기준, 연습 레인 입수 기준)
  - 수영장 시계가 약 2분 빠름 → 실제 7시 58분까지 도착 필요
- **지각 벌금**: 1분당 500원 (최대 1만원)
- **무단 불참**: 1만원 (금요일 자정까지 불참 표시 시 면제)
- 벌금은 당일 즐수팀 계좌로 자진 입금

## 출석체크 방법

불참 시 **금요일 자정까지** 출석체크에 불참 표시 필수

1. 카카오톡방 오른쪽 상단 **☰** 메뉴버튼 터치
2. **☑ 일정** 터치
3. 날짜 선택 후 참석 여부 선택

- 채팅으로만 남기면 인원 파악이 어려움 → 꼭 일정 기능 사용
- 미응답 시 자동 참석으로 간주

## 신입 회원 안내

### 첫 참석
- 첫 1회는 **체험** 가능 (무료)
- 2회차부터 가입 필수

### 수모 구입 (필수)
- 1장: 2만원 / 2장: 3만원(장당 1만 5천원)
- 회비와 별도로 즐수팀 계좌로 입금
- 입금 후 최선숙 회원님께 수령

### 첫 입금 금액
- **수모 2장 권장**: 5만원 (회비 2만 + 수모 3만)
- **수모 1장**: 4만원 (회비 2만 + 수모 2만)
- 가입 승인 시 바로 입금, 다음 달부터 매월 1일 납부

### 가입 후 할 일
1. **입금**: 회비 + 수모 금액을 즐수팀 계좌로 입금
2. **카톡방 참여**: 관리자에게 초대 요청
3. **수모 수령**: 최선숙 회원님께 수령
