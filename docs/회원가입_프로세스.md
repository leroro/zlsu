## 회원가입 프로세스 문서 (기초 정보)

### 1단계: 가입 신청

| 항목 | 내용 |
|------|------|
| **누가** | 신규 가입 희망자 |
| **뭘** | 가입 신청서 작성 |
| **어떻게** | 기존 회원에게 가입 신청 링크를 받아서 접속 → 회칙 확인 및 동의 → 신청서 작성 (추천인 = 링크 전달한 회원) |
| **결과** | 가입 신청 완료, 승인 대기 상태 |

---

### 2단계: 가입 승인

| 항목 | 내용 |
|------|------|
| **누가** | 총무 |
| **뭘** | 가입 조건 확인 및 승인/반려 처리 |
| **어떻게** | 관리자 시스템에서 신청 내역 확인 → 가입 조건 충족 여부 검토 → 승인 또는 반려 |
| **결과** | 승인 시 다음 단계 진행, 반려 시 사유 안내 |

---

### 3단계: 가입비 납부 안내

| 항목 | 내용 |
|------|------|
| **누가** | 총무 |
| **뭘** | 가입비 납부 안내 |
| **어떻게** | 승인된 회원에게 연락 (가입 신청 시 입력한 연락처 활용) → 가입비 안내 (첫 달 회비 2만원 + 수모 1장 2만원 = 총 4만원) → 입금 계좌 안내 (회칙에 명시된 카카오뱅크 임미선 계좌) |
| **결과** | 회원이 가입비 입금 |

---

### 4단계: 입금 확인

| 항목 | 내용 |
|------|------|
| **누가** | 총무 |
| **뭘** | 가입비 입금 확인 |
| **어떻게** | 계좌 입금 내역 확인 → 입금자명과 신청자 매칭 |
| **결과** | 입금 확인 완료 |

---

### 5단계: 카카오톡 단톡방 초대

| 항목 | 내용 |
|------|------|
| **누가** | 총무 |
| **뭘** | 카카오톡 단톡방 초대 |
| **어떻게** | 가입 신청 시 입력한 연락처로 카카오톡 친구 추가 → 동호회 단톡방에 초대 |
| **결과** | 단톡방 참여 완료 |

---

### 6단계: 카카오뱅크 모임통장 초대

| 항목 | 내용 |
|------|------|
| **누가** | 총무 |
| **뭘** | 카카오뱅크 모임통장 멤버 초대 |
| **어떻게** | 카카오뱅크 앱에서 모임통장 멤버로 초대 (회원은 카카오뱅크 앱 설치 및 가입 필요) |
| **결과** | 모임통장 멤버 등록 완료, 이후 월 회비는 모임통장으로 관리 |

---

### 7단계: 수모 수령

| 항목 | 내용 |
|------|------|
| **누가** | 수모 담당자 (최선수 회원) |
| **뭘** | 수모 지급 |
| **어떻게** | 총무가 수모 담당자에게 신규 회원 정보 전달 → 수모 담당자가 신규 회원에게 수모 지급 |
| **결과** | 수모 수령 완료 |
| **비고** | ⚠️ 수모 지급 프로세스 세부 사항은 보류 중 (시스템화 여부 검토 필요) |

---

### 8단계: 첫 참석

| 항목 | 내용 |
|------|------|
| **누가** | 신규 회원 |
| **뭘** | 토요일 수영 참석 |
| **어떻게** | 수령한 수모 착용 → 토요일 수영장 방문 → 동호회 활동 시작! |
| **결과** | 정식 회원으로 활동 시작 🏊 |

---

## 역할별 요약

| 역할 | 담당 업무 |
|------|----------|
| **신규 가입자** | 가입 신청서 작성, 가입비 납부, 수모 수령, 첫 참석 |
| **추천인 (기존 회원)** | 가입 신청 링크 전달 |
| **총무** | 가입 승인/반려, 가입비 안내 및 입금 확인, 카톡 초대, 카뱅 모임통장 초대, 수모 담당자에게 정보 전달 |
| **수모 담당자 (최선수 회원)** | 수모 지급 |
