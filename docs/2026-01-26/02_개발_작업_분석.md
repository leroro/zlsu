## 핵심 개념 정리

### 정원 개념

| 구분 | 정의 | 제한 | 시스템 설정 |
|------|------|------|------------|
| **활동 회원 정원** | 회비 내고 참석 의사 있는 회원 | 16명 | `maxCapacity` (기존) |
| **주간 참석 정원** | 해당 주에 실제 수영하는 인원 | 14명 | `weeklyCapacity` (신규) |

### 활동/휴면 제도

| 구분 | 활동 회원 | 휴면 회원 |
|------|----------|----------|
| 정의 | 이번 달 1회 이상 참석 의사 | 이번 달 참석 안 함 |
| 회비 | 납부 (2만원) | 면제 |
| 참석 | 가능 | 불가 |
| 기본값 | ✅ (신청 안 하면 활동) | 신청 필요 |

**운영 방식:**
- 매월 1일 = 회비 납부일 = 휴면 신청 마감일
- 매월 10일 = 미응답자 강제 휴면 처리
- 휴면 중 복귀 = 월 중 회비 입금 → 사이트에서 활동 전환 신청

### 매월 10일 총무 체크리스트

```
□ 회비 납부 확인
  - 납부 완료 → 활동 유지
  - 미납부 + 휴면 신청 → 휴면 처리
  - 미납부 + 무응답 → 연락 후 강제 휴면 처리

□ 사이트 상태 업데이트
  - 휴면 신청자 → 휴면으로 변경
  - 강제 휴면자 → 휴면으로 변경
```
