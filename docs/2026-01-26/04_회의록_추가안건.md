# 2026.02.01 티타임 - 추가 논의 사항

지난번 공유드린 내용에서 추가로 정리된 부분만 모았습니다.

---

## 1. 월간 운영 일정 (제안)

| 일자 | 내용 |
|------|------|
| 매월 1일 | 알림 발송 (회비 납부 / 휴면 신청 안내) |
| 매월 1~9일 | 회원들 납부 또는 휴면 신청 |
| 매월 10일 | 총무가 미응답자 휴면 처리 |

**알림 예시 (캘린더 반복 설정)**
```
📢 [즐수팀] 2월 회비 납부일입니다!

✅ 활동하실 분: 회비 2만원 납부
😴 휴면하실 분: 사이트에서 휴면 신청

⏰ 마감: 2월 9일까지
   (10일에 미응답자는 휴면 처리됩니다)
```

### 지난 투표에서 나온 의견

> "활동정원이나 대기문제는 다음주에 다른 분들 다른 의견도 들어봐서 얘기해보면 좋을 것 같습니다" - 병돈님

투표에서 10명 모두 동의했지만, **추가 논의가 필요하다**는 의견이 있었어요.

### 휴면 기간에 따른 정원 처리 (제안)

단기 휴면인데 그 사이에 자리가 차버리면 억울하잖아요. 그래서:

| 휴면 기간 | 정원 처리 | 설명 |
|----------|----------|------|
| **단기 (1~2개월)** | 자리 유지 | 잠깐 쉬는 거니까 정원에 포함 |
| **장기 (3개월+)** | 자리 양보 | 정원에서 제외, 신규 가입 가능 |

- 단기 휴면자는 안심하고 쉴 수 있음
- 장기 휴면자만 자리를 양보

### 논의 포인트
- [ ] 활동/휴면 제도, 이대로 괜찮을까요? 다른 의견 있으신가요?
- [ ] 단기/장기 기준을 2개월로 할까요, 3개월로 할까요?
- [ ] 월간 운영 일정 (1일 알림, 10일 정리) 괜찮을까요?
- [ ] 휴면 신청 기간을 더 여유 있게 할까요? (예: 전월 25일까지)

## 2. 정원 기준 정리

### 정원 구조 (제안)

| 구분 | 인원 |
|------|------|
| **레인 수용** | 코치 포함 15명 |
| **활동 회원 정원** | 14명 (코치 제외) |
| **남는 자리** | 코치님 재량으로 게스트 참여 |

- 현재 16명 중 휴면 1명 이상 있으니 14명이면 수용 가능
- 휴면자 수 파악 → 신규 가입 가능 여부 판단

### 스탭 정원 별도 관리 (제안)

**제안 원칙:** 고정 업무 봉사자 정원 어드밴티지

- 코치님은 수영 지도 봉사로 자리가 항상 확보되어 있어요.
- 총무(저)도 고정 업무(톡방/회비/안건/출결/이벤트/사이트 관리)로 시간 투자가 필요한 만큼,
총무 자리도 항상 비워두면 어떨지 의견을 구해봅니다. ^^

**총무 자리를 비워둘 경우:**

| 구분 | 인원 |
|------|------|
| 일반 회원 정원 | 13명 |
| 총무 자리 (항상 확보) | 1명 |
| 코치님 | 1명 |
| **합계 (레인 수용)** | **15명** |

### 논의 포인트
- [ ] 총무 자리 항상 확보, 괜찮을까요?

## 3. 역할 분담 관련 (총무 의견)

> "팀 사이트 개발이나 자동화, 온라인 관리 같은 부분은 제가 기쁘게 계속 맡아서 할 수 있어요! 개발자로서 이런 기여를 할 수 있어서 좋습니다.
>
> 다만, 행사 준비나 현장 진행 같은 오프라인 업무는 혼자 감당하기 부담이 될 수 있어서, 이런 부분은 함께 분담하면 더 좋을 것 같아요.
>
> 각자 잘할 수 있는 영역에서 기여하면서, 팀 전체가 함께 운영하는 느낌으로 가면 좋겠습니다."

## 4. 수모 착용 관련 (진행 방향)

지난 투표 결과와 논의를 바탕으로 정리했어요.

### 진행 순서

**1단계: 당장은 메시 수모 착용 (2월 중)**
- 실리콘 제작 전까지는 기존 메시 수모 착용 부탁드려요
- 팀 통일성을 위해 꼭 써주세요!

**2단계: 메시 수모 추가 지급**
- 기존에 메시 수모 구매하신 분들께 동일 수량 추가 지급
- 예: 1장 사신 분 → 1장 더, 3장 사신 분 → 3장 더
- 물빠짐 문제 보완 + 재고 소진 효과

**3단계: 실리콘 수모 제작 (2월 중 주문)**
- 주문제작 최소 수량: 30장 이상
- 예산: 수십만 원 필요 → 2월 회비 수금 후 진행
- 제작 기간: 주문 후 2~3주 소요 예상

**4단계: 혼용 허용 (실리콘 도착 후)**
- 메시/실리콘 선택 가능
- 단, **색상은 동일하게** 맞춰서 통일성 유지

### 논의 포인트
- [ ] 메시/실리콘 혼용 허용, 괜찮을까요?
- [ ] 기존 구매자 추가 지급 방식, 괜찮을까요?
- [ ] 실리콘 수모 색상은 어떤 색으로 할까요?

## 이번 회의에서 추가로 결정할 것

| 안건 | 비고 |
|------|------|
| 활동/휴면 제도 | 추가 의견 확인 |
| 휴면 기간 기준 | 단기(자리유지) vs 장기(양보) 기준 |
| 월간 운영 일정 | 1일 알림 / 10일 정리 |
| 정원 기준 | 활동 14명 / 레인 코치포함 15명 / 게스트 |
| 총무 자리 확보 | 일반 13명 + 총무 + 코치 = 15명 |
| 역할 분담 | 온라인 vs 오프라인 |
| 수모 착용 | 메시 착용 + 실리콘 제작 + 혼용 |
