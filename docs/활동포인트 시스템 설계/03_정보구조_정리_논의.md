## 문제점

1. **"활동지수란?" 페이지 정체성 불명확**
   - 활동지수 설명만 있으면 제목이 맞음
   - 하지만 주종목, 생일, 대회, 수모 등 기타 안내도 포함됨
   - 제목과 내용이 안 맞음

2. **운영 가이드 vs 회원 안내 구분 불명확**
   - 운영 가이드: 스태프/운영진 대상 업무 가이드
   - 하지만 일반 회원도 접근 가능한 상태

3. **메뉴 증가로 인한 복잡성**
   - 메뉴가 점점 늘어나고 있음
   - 계층적 구조 또는 그룹핑 필요

## 정리 방안 (논의 필요)

### 방안 1: 페이지 통합 + 이름 변경

"활동지수란?" → "회원 안내" 또는 "알아두면 좋은 것들"로 변경
- 활동지수
- 주종목 배지
- 생일 축하
- 대회 참가
- 수모 구입

운영 가이드는 그대로 유지 (스태프 대상)

### 방안 2: 계층적 메뉴 구조

```
📖 안내
  ├── 회원 안내 (일반 회원 대상)
  ├── 운영 가이드 (스태프 대상)
  └── 회칙
```

### 방안 3: 운영 가이드 접근 제한

- 운영 가이드를 스태프만 볼 수 있게 제한
- 또는 관리자 메뉴로 이동
