# 다음 개발 작업 목록

1. 가입 절차 업데이트
   - 총무가 초대하는게 아니라 로그인 후 카톡방 입장하기 버튼 클릭해야 함.

2. 최초 가입 승인 이후 안내 단계 필요
   - 대시보드 할 일 섹션에 팀 카톡방에 입장하라는 플로우 추가해야하지 않을지?

3. 가입 안내 화면에 CTA 버튼 플로팅 처리
   - 내용이 길어서 가입 신청하기 버튼이 하단에 상시 고정되게 처리하면 더 좋겠음. 맨 하단에 내려가면 원래 UI대로 유지되고.

4. 음력 회원 생일은 생일 명단에서 양력 기준으로 표시되도록 개선
   - 예를들어 정상옥님 생일을 음력 2/22로 세팅하고, 해당 회원의 생일이 4/9(목)으로 나오고 있는지 확인 필요
   - 음력 생일인 사람은 따로 표시도 되어야 할 것 같음


# 검토 필요 건
1. [논의] 생일에 가장 가까운 토요일 날짜 출력 기능
   - 이건 바로 개발하지 말고 아이디어 회의만 하고 싶음. 축하날짜를 언제로 할지 자동으로 알려주는 로직도 있으면 너무 편리할 것 같음. 단, 간혹 한달에 두명 이상의 회원이 있으면 묶어서 같이 축하해주기도 하므로 이 부분은 딱 로직으로 잡을 수는 없어서 좋은 방안 검토 필요

2. 사람들이랑 찍은 사진도 추가하면 좋을듯 함.
   - 단, 업로드 기능 같은건 제공하지말고 그냥 상징적인 활동 사진 몇개만 대표적으로 넣으려고 함. 예를들어 대회 나간거랑, 송년회 멋지게 한 모습 같은거. 어디에 넣을까? 소개?
