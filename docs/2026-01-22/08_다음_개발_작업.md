## 1. 신규 회원 환영 메시지 및 카톡방 입장 안내

### 요구사항
- 가입 승인 완료 후 첫 로그인 시 환영 메시지 표시
- 카카오톡 단톡방 입장 버튼 클릭 유도
- 한 번 입장 버튼 클릭 후에는 메시지 숨김

### 상세 내용
1. **표시 조건**
   - 회원 상태가 `active`
   - 아직 카톡방 입장 버튼을 클릭하지 않은 경우 (새 플래그 필요)

2. **UI 위치**
   - HomePage 상단 (할 일 섹션보다 위)

3. **메시지 내용**
   ```
   🎉 가입을 환영합니다!
   카카오 단톡방 버튼을 클릭해서 입장해 주세요.
   [단톡방 입장하기] 버튼
   ```

4. **구현 방법**
   - Member 인터페이스에 `hasJoinedKakao?: boolean` 필드 추가
   - 버튼 클릭 시 해당 필드를 `true`로 업데이트
   - `true`인 경우 환영 메시지 숨김

### 관련 파일
- `src/lib/types.ts` - Member 인터페이스 수정
- `src/lib/api.ts` - 카톡방 입장 완료 API 함수 추가
- `src/pages/HomePage.tsx` - 환영 메시지 UI 추가

## 2. 운영 가이드 문서 업데이트

### 요구사항
현재 구현된 시스템 기준으로 운영 가이드 문서 수정

### 수정 사항

#### 가입 처리 절차 (총무용) 수정
**기존:**
```
4. 카카오톡 단톡방 초대
   - 신청서에 기재된 연락처로 카톡 친구 추가
   - 즐수팀 단톡방에 초대
```

**변경:**
```
4. 카카오톡 단톡방 안내
   - 시스템에서 가입 승인 시 자동으로 단톡방 입장 버튼 활성화
   - 신규 회원이 직접 버튼 클릭하여 입장
```

#### 추천인 역할 수정
**추가:**
```
### 2. 수모 전달
- 수모 담당자(최선숙 회원)에게 수모 수령
- 신규 회원에게 수모 전달
※ 추천인 동의 시 "수모 전달" 체크박스 확인 필수
```

### 관련 파일
- `public/rules/operations-guide.md`

## 3. 추천인 동의 페이지 수모 체크박스 문구 개선

### 현재
```
가입 완료 후 수모를 전달하겠습니다.
```

```
수모 담당자(최선숙 회원)에게 수모를 수령하여 신규 회원에게 전달하겠습니다.
```
