#2 사이트 기본 언어 설정을 한글로 변경

## 이슈 유형
- 버그
- 설정 수정

## 배경
사이트 UI 및 콘텐츠는 한글로 제공되고 있으나  
시스템 기본 언어(language) 설정이 영문으로 되어 있음  
이로 인해 기본 세팅, 일부 메시지 또는 확장 기능에서 언어 불일치 가능성 존재

## 수정 내용
- 시스템 기본 language 설정을 영문에서 한글로 변경
- 초기 로딩 및 기본 환경에서 한글이 기본값으로 적용되도록 설정 수정

## 처리 기준
- 기능 개발 없음
- 글로벌 설정 값 수정에 해당
- 기존 한글 콘텐츠 및 UI 동작에 영향 없도록 확인 필요

## 비고
한글 사이트 기준에 맞는 기본 언어 일관성 확보 목적
