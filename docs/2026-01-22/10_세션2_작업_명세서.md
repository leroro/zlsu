## 남은 작업 목록

### 1. 가입 완료 화면 안내문 수정

**현재 문제:** 예전 안내문이 그대로 남아있음

**올바른 안내 내용:**
```
가입 신청이 완료되었습니다.

다음 절차로 진행됩니다:
1. 추천인이 동의해야 합니다
2. 추천인 동의 후 관리자가 입금 확인합니다
3. 입금 확인 후 최종 가입 승인됩니다

가입비 납부 안내는 관리자 승인 단계에서 별도 연락드립니다.
```

**관련 파일:** `src/pages/ApplyPage.tsx` 또는 가입 완료 화면 컴포넌트

---

### 2. 음력 생일 양력 변환 표시

**요구사항:**
- 음력으로 등록된 생일을 양력 기준으로 변환하여 표시
- 예: 정상옥님 음력 2/22 → 양력 4/9(목) 으로 표시
- 음력 생일인 사람은 별도 표시 (예: "(음)" 또는 아이콘)

**구현 방법:**
1. 음력→양력 변환 라이브러리 사용 (예: `korean-lunar-calendar`)
2. 생일 목록 표시 시 음력 여부 확인 후 변환
3. UI에 음력 표시 추가

**참고:** npm 패키지 설치가 필요할 수 있음
```bash
npm install korean-lunar-calendar
```

**관련 파일:**
- `src/pages/MembersPage.tsx` (생일 탭)
- `src/pages/HomePage.tsx` (생일 섹션) - 세션 1 완료 후 수정 가능
