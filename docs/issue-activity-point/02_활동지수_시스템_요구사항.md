# 활동 지수 시스템 요구사항

## 배경

현재 시스템은 회원의 **수영 실력**(초급/중급/상급/마스터)을 기준으로 프로필 이모지를 표시합니다.

### 문제점
- 수영을 잘하는 신입 회원이 바로 높은 레벨(🦈)로 표시됨
- 모임의 취지는 **친목**이며, 활동을 열심히 하는 분이 높게 보여야 함
- 단순 수영 실력만으로는 모임 기여도를 반영할 수 없음

## 요구사항

### 1. 수영 수준 정보 유지
- 가입 시 선택하는 수영 수준(초급/중급/상급/마스터)은 **그대로 유지**
- 회원 정보에 저장되어 참고용으로 사용
- 대회 경험 필드도 별도 유지

### 2. 활동 지수 도입
- 프로필에 표시되는 이모지를 **활동 지수** 기준으로 변경
- 활동 지수는 **관리자가 직접 지정**
- 신규 회원은 기본 활동 지수(가장 낮은 단계)로 시작

### 3. 활동 지수 단계 (안)

| 레벨 | 아이콘 | 명칭 | 기준 |
|------|--------|------|------|
| 1 | 🛟 | 뉴비 | 가입 후 2개월까지 |
| 2 | 🏊 | 일반 | 월 2회 정도 참여 |
| 3 | 🐬 | 열정 | 월 3회 이상 참여 |
| 4 | 🦈 | 핵심 | 월 3회 이상 참여 + 대회 꾸준히 참여 |
| 5 | 🏆 | 스태프 | 모임 운영 업무/기여 |

## 기능 명세

### Member 인터페이스 변경

```typescript
interface Member {
  // 기존 필드
  swimmingLevel?: SwimmingLevel;  // 수영 수준 (참고용, 그대로 유지)
  competitionHistory?: CompetitionHistory;  // 대회 경험 (그대로 유지)

  // 새 필드
  activityLevel?: ActivityLevel;  // 활동 지수 (관리자 지정)
}

// 활동 지수 타입
type ActivityLevel = 'newbie' | 'regular' | 'active' | 'core' | 'legend';
```

### UI 변경사항

#### 1. 회원 명단 (MembersPage.tsx)
- 기존: 수영 레벨 이모지 표시
- 변경: **활동 지수 이모지** 표시
- 수영 레벨은 별도 텍스트로 표시 (옵션)

#### 2. 관리자 페이지
- 회원별 활동 지수 변경 기능 추가
- 드롭다운 또는 버튼으로 레벨 조정

#### 3. 가입 신청 (ApplyPage.tsx)
- 수영 수준 선택은 **그대로 유지**
- 활동 지수는 가입 시 선택하지 않음 (자동으로 'newbie')

#### 4. 마이페이지
- 본인의 활동 지수 확인 가능
- 수정은 불가 (관리자만 가능)

## 개발 계획

### Phase 1: 타입 및 데이터 구조 변경
1. `types.ts`: ActivityLevel 타입 추가, Member 인터페이스에 activityLevel 필드 추가
2. `constants.ts`: ACTIVITY_LEVELS, ACTIVITY_LEVEL_EMOJIS, ACTIVITY_LEVEL_LABELS 추가
3. `mockData.ts`: 테스트 데이터에 activityLevel 추가

### Phase 2: API 함수 추가
1. `api.ts`: updateMemberActivityLevel(memberId, level) 함수 추가

### Phase 3: UI 변경
1. `MembersPage.tsx`: 이모지 표시 로직을 swimmingLevel → activityLevel로 변경
2. `AdminMembersPage.tsx` (신규): 회원별 활동 지수 관리 UI
3. `HomePage.tsx`: 대시보드에 활동 지수 표시 (옵션)

### Phase 4: 가입/마이페이지
1. `ApplyPage.tsx`: 수영 수준 필드 유지, 활동 지수 관련 변경 없음
2. `ProfilePage.tsx`: 활동 지수 표시 추가

**페이지명**: "즐수팀 가이드" 또는 "알아두면 좋은 것들"

#### 1. 즐수팀 활동지수

| 레벨 | 아이콘 | 명칭 | 기준 |
|------|--------|------|------|
| 1 | 🛟 | 뉴비 | 가입 후 2개월까지 |
| 2 | 🏊 | 일반 | 월 2회 정도 참여 |
