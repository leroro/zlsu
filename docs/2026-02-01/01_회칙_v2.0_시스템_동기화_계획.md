## 3. 수정 계획

### 3.1 회칙 파일 통합 (중복 제거)

**현재 문제**: `rules.md`와 `v2.0.md`가 동일한 내용으로 중복 관리

**해결 방안**:
- `rules.md` 삭제
- `TERMS.RULES`가 `RULES_VERSIONS[0]` (최신 버전)을 자동 참조하도록 변경

**변경 전**:
```typescript
export const TERMS = {
  RULES: '/rules/rules.md',  // 별도 파일
};
```

**변경 후**:
```typescript
export const TERMS = {
  RULES: RULES_VERSIONS[0].path,  // 항상 최신 버전 자동 참조
};
```

**효과**:
- 새 버전 추가 시 `RULES_VERSIONS` 배열에만 추가하면 됨
- 가입 약관도 자동으로 최신 버전 노출
- 문서 1개만 관리

**삭제 파일**:
- `public/rules/rules.md`
- `dist/rules/rules.md`
