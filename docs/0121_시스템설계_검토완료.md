## 2. 개선 필요 항목

### 2.1 가입신청서 개선

현재 가입신청서에 없는 필드:

| 추가 필드 | 설명 |
|----------|-----|
| 이용약관(회칙) 동의 | 체크박스 + 전문 보기 |
| 개인정보수집 동의 | 체크박스 + 전문 보기 |
| 추천인 | 텍스트 입력 (선택) |
| 수영 실력 | 자유형/배영/평영/접영 가능 여부 체크 |

### 2.2 약관 통합 관리

- 회칙(`/rules`)과 가입신청 화면의 약관을 동일한 파일에서 불러오도록 통합
- 개인정보수집동의 문서 추가 필요

### 2.3 탈퇴 기능

- 현재 탈퇴 신청 기능이 없음
- MyPage에 "탈퇴 신청" 버튼 추가 필요

## 3. 확인 필요 사항

**[Q1] 탈퇴 처리 방식**
- A) 사용자가 탈퇴 신청 → 관리자 승인 후 탈퇴
- B) 사용자가 탈퇴 신청 → 즉시 탈퇴 처리
- C) 관리자만 탈퇴 처리 가능
=> 사용자가 탈퇴 신청 → 관리자 승인 후 탈퇴하는 기능과, 관리자가 직접 강퇴시키는 기능도 필요합니다.
근데 승인거절도 있어야하지 않아요? 거절 클릭 후에 얼럿으로 한번 더 확인하구요.
탈퇴 얼럿도 필요하고요. 이미 납부된 회비는 반환되지 않는다는 안내문구 필요함. 이 내용은 약관에도 들어가야하구요.

**[Q2] 수영 실력 입력 형태**
- A) 영법별 체크박스 (가능한 영법만 체크)
- B) 영법별 수준 선택 (못함/초급/중급/상급)
- C) 자유 텍스트
=> 영법별 체크하는게 좋겠어요. 어디까지 배우셨냐는 톤으로.

**[Q3] 개인정보수집동의 문구**
- 초안이 있으신가요? 없으면 기본 템플릿으로 작성해드릴까요?
=> 기본 템플릿으로 작성, 필요시 직접 수정하겠음
