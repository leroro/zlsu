# 다음 작업: AboutPage / InfoPage 공통 컴포넌트 분리

## 배경

AboutPage(회원용)와 InfoPage(비회원용)의 본문 내용이 동일한데 각각 별도로 관리되고 있음.
내용 수정 시 양쪽을 모두 업데이트해야 하는 비효율 발생.

**실제 문제:** 2026-02-08 세션에서 InfoPage만 수정하고 AboutPage는 업데이트하지 못함.
- AboutPage에 실리콘 수모 안내 없음
- 수모 가격 구 버전 (메쉬 2장 3만원)
- "메가커피" → "근처 카페" 미반영
- 수모 미착용 벌금 FAQ 누락
- 정원 마감 보조 텍스트 없음

## 목표 구조

```
src/components/about/
├── HeroSection.tsx           히어로
├── ActivitySection.tsx       활동 소개
├── EventSection.tsx          이벤트 및 대외 활동
├── RecommendSection.tsx      이런 분께 추천
├── JoinProcessSection.tsx    가입 절차
├── JoinInfoSection.tsx       가입 안내 (정원/회비/수모/납부금액)
└── FaqSection.tsx            자주 묻는 질문

src/pages/
├── AboutPage.tsx    → 공통 컴포넌트 + 가입신청 CTA
└── InfoPage.tsx     → 공통 컴포넌트 + 회칙보기 CTA
```

## 차이점 (개별 처리 필요)

| 항목 | AboutPage (회원용) | InfoPage (비회원용) |
|------|-------------------|-------------------|
| 래퍼 | `max-w-2xl mx-auto` (앱 레이아웃 내) | `min-h-screen bg-gray-100` > `max-w-2xl mx-auto py-4` (독립) |
| CTA 내용 | 가입 신청 + 회칙 보기 | 회칙 자세히 보기만 |
| 플로팅 CTA | 가입 신청하기 (공석 있을 때만) | 회칙 자세히 보기 |
| 추천인 파라미터 | `?ref=` 파라미터로 가입 URL 생성 | 없음 |
| 플로팅 트리거 | joinInfoSection 진입 ~ CTA 도달 | 동일 |
| CTA 링크 경로 | `/rules`, `/apply` | `/info/rules` |

## 구현 시 주의사항

1. **InfoPage가 최신 버전** — InfoPage 기준으로 공통 컴포넌트 추출
2. **JoinInfoSection에 ref 전달 필요** — 플로팅 CTA 트리거용 `joinInfoSectionRef`
3. **CTA 섹션에 ref 전달 필요** — `ctaSectionRef`
4. **document.title** — InfoPage는 직접 설정, AboutPage는 useDocumentTitle 훅 사용
5. **가입 안내의 정원/회비 데이터** — `getActiveAndInactiveMemberCount()`, `getSettings()` 호출은 페이지 레벨에서 하고 props로 전달하거나, 각 섹션 컴포넌트 내에서 직접 호출
