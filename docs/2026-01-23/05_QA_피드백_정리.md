## 논의 필요한 항목

### 4. 회칙 숙지 별도 섹션 분리
- **현재**: 체크리스트에 "회칙 숙지" 포함
- **요청**:
  - 회칙을 별도 섹션으로 분리
  - 레이어 팝업으로 회칙 전문 표시
  - 스크롤 끝까지 내려야 "확인" 버튼 활성화
  - "회칙 전문을 확인했습니다" + "동의합니다" 체크
- **이유**: 단순 이용약관이 아니라 회비 등 중요 내용 포함

### 5. 가입 신청 완료 화면 간소화
- **현재**:
  - "추천 동의를 요청해 주세요" 메시지
  - "앞으로의 절차" (추천인 동의 → 관리자 승인 → 가입 완료)
  - "문의사항은 OO님에게 연락해 주세요"
  - 중복되는 느낌
- **요청**: 짜임새 있게 재구성 (추천 동의 요청 메시지는 유지)

### 6. 화면 이동 시 스크롤 위치 문제 (iOS Safari)
- **현상**: 페이지 이동 시 스크롤이 맨 위로 안 올라가고 중간에 위치
- **환경**: 아이폰 Safari
- **추정 원인**: 모바일 브라우저 주소창 높이 변화, 고정 헤더
- **비고**: 이전에 두 번 수정 요청했으나 여전히 발생

### 7. 가입비 납부 안내 노출 시점
- **현재**: 관리자 승인 대기 단계에서만 노출
- **요청**: 추천인 동의 대기 단계부터 노출 (항상 보여야 함)
- **이유**: 가입비 입금 확인 후 관리자가 승인하는 구조

### 8. 대시보드 온보딩 섹션 검토
- **현재**: "다음 단계를 완료해주세요!" 섹션 존재
- **질문**: 이 온보딩 단계가 꼭 필요한지?
- **대안**: 카카오 채팅방 자동 안내 메시지로 대체 가능

### 9. 카카오 채팅방 자동 안내 메시지
- **요청**: 채팅방 입장 시 자동으로 보내는 안내 메시지 안 제공
- **용도**: 신규 회원이 채팅방에 들어왔을 때 필요한 정보 안내
