| 6 | iOS Safari 스크롤 문제 | useLayoutEffect + requestAnimationFrame + setTimeout(50ms) 적용 | 모바일 테스트 대기 중 |

**수정 파일**: `src/components/common/ScrollToTop.tsx`
```tsx
useLayoutEffect(() => {
  const scrollToTop = () => {
    window.scrollTo({ top: 0, left: 0, behavior: 'instant' });
    document.documentElement.scrollTop = 0;
    document.body.scrollTop = 0;
  };
  scrollToTop();
  requestAnimationFrame(() => {
    scrollToTop();
    setTimeout(scrollToTop, 50);
  });
}, [pathname]);
```
