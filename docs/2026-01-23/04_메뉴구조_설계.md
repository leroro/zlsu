# 메뉴 구조 설계

## 사용자 상태 분류

| 상태 | 설명 |
|------|------|
| 비로그인 | 로그인하지 않은 방문자 |
| 승인대기 (pending) | 가입 신청 후 승인 전 |
| 일반회원 (active/inactive) | 승인 완료된 활성/휴면 회원 |
| 관리자 (admin) | 관리자 권한 |

---

## 풀 메뉴 구조 (햄버거 메뉴)

```
┌─────────────────────────┐
│ [로그인] / [로그아웃]   │  ← 인증 영역
├─────────────────────────┤
│                         │
│  === 모임 활동 ===           │
│  🏠 홈                  │
│  🏊 소개                │
│  📖 가이드              │
│  👥 회원 명단           │
│                         │
├─────────────────────────┤
│                         │
│  === 운영 규정 ===           │
│  📋 운영자 가이드       │
│  📜 회칙                │
│                         │
├─────────────────────────┤
│                         │
│  === 관리자 ===         │
│  📋 신청 관리           │
│  👥 회원 관리           │
│  ⚙️ 설정                │
│                         │
└─────────────────────────┘
```

---

## 상태별 메뉴 노출

| 메뉴 | 비로그인 | 승인대기 | 일반회원 | 관리자 |
|------|:--------:|:--------:|:--------:|:------:|
| **인증 영역** |||||
| 로그인 | ✅ | - | - | - |
| 로그아웃 | - | ✅ | ✅ | ✅ |
| **모임 활동** |||||
| 홈 | ✅ | ✅ | ✅ | ✅ |
| 소개 | ✅ | ✅ | ✅ | - |
| 가이드 | - | - | ✅ | - |
| 회원 명단 | - | - | ✅ | - |
| 가입 신청 | ✅ (정원 여유시) | - | - | - |
| **운영 규정** |||||
| 운영자 가이드 | - | - | ✅ | - |
| 회칙 | ✅ | ✅ | ✅ | ✅ |
| **관리자** |||||
| 신청 관리 | - | - | - | ✅ |
| 회원 관리 | - | - | - | ✅ |
| 설정 | - | - | - | ✅ |

## 상태별 상세 메뉴

### 1. 비로그인

```
┌─────────────────────────┐
│ [로그인]                │
├─────────────────────────┤
│  🏠 홈                  │
│  🏊 소개                │
│  ✍️ 가입 신청           │  ← 정원 마감 시 비활성화
├─────────────────────────┤
│  📜 회칙                │
└─────────────────────────┘
```

### 2. 승인대기 (pending)

```
┌─────────────────────────┐
│ [로그아웃]              │
├─────────────────────────┤
│  🏠 홈                  │  ← 승인 상태 확인
│  🏊 소개                │
├─────────────────────────┤
│  📜 회칙                │
└─────────────────────────┘
```

### 3. 일반회원 (active/inactive)

```
┌─────────────────────────┐
│ [로그아웃]              │
├─────────────────────────┤
│  🏠 홈                  │
│  🏊 소개                │
│  📖 가이드              │  ← "필독! 모임 가이드"
│  👥 회원 명단           │
├─────────────────────────┤
│  📋 운영자 가이드       │
│  📜 회칙                │
└─────────────────────────┘
```

### 4. 관리자

```
┌─────────────────────────┐
│ [로그아웃]              │
├─────────────────────────┤
│  🏠 홈                  │
│  📋 신청 관리           │
│  👥 회원 관리           │
│  ⚙️ 설정                │
├─────────────────────────┤
│  📜 회칙                │
└─────────────────────────┘
```

## 데스크톱 상단 메뉴

데스크톱에서는 햄버거 메뉴 대신 상단 바에 주요 메뉴 노출

### 비로그인
```
[즐수팀 로고]     [소개] [회칙] [가입신청]     [로그인]
```

### 일반회원
```
[즐수팀 로고]     [소개] [가이드] [회원명단]     [로그아웃]
```

### 관리자
```
[즐수팀 로고]     [신청관리] [회원관리] [설정]     [로그아웃]
```

---

## 변경 포인트 요약

1. **로그인/로그아웃 버튼**: 맨 위로 이동
2. **메뉴 섹션화**: 모임 활동 / 운영 규정 / 관리 구분
3. **메뉴 이름 변경**:
   - 모임 소개 → 소개
   - 활동지수란? → 가이드 (필독! 모임 가이드)
   - 운영 가이드 → 운영자 가이드
4. **메뉴 사이즈**: 컴팩트하게 줄임
5. **상태별 노출 제어**: 위 표 기준
