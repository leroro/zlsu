### 작업 2: 음력 생일 양력 변환 표시

**파일:**
- `src/pages/MembersPage.tsx` (생일 탭)
- 필요시 `src/lib/api.ts`에 유틸 함수 추가

**요구사항:**
1. 음력으로 등록된 생일을 해당 연도 양력으로 변환하여 표시
2. 음력 생일인 경우 "(음)" 표시 추가
3. 예: `정상옥 4/9 (음)` (음력 2/22가 올해 양력 4/9인 경우)

**구현 방법:**
```bash
npm install korean-lunar-calendar
```

```typescript
import { KoreanLunarCalendar } from 'korean-lunar-calendar';

function lunarToSolar(lunarMonth: number, lunarDay: number, year: number) {
  const calendar = new KoreanLunarCalendar();
  calendar.setLunarDate(year, lunarMonth, lunarDay, false);
  return {
    month: calendar.getSolarMonth(),
    day: calendar.getSolarDay()
  };
}
```
