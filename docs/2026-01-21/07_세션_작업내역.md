## 완료된 작업

### 1. 홈페이지 UX 플로우 개선
- **CTA 버튼 순서 변경**: [로그인] [모임 소개 보기] 순서로 배치
- **가입 신청 링크**: "아직 회원이 아니신가요? 가입 신청하기" 텍스트 링크로 변경
- **회칙 확인하기 링크 제거**: 가입 시 자동으로 보여주므로 불필요
- **정원 마감 시**: "현재 정원이 마감되었습니다" 메시지 표시

### 2. 모바일 레이아웃 개선
- **패딩 제거**: 모바일에서 상/하단 콘텐츠 패딩 제거 (`App.tsx`)
- **뒤로가기 버튼**: 모바일 헤더에 뒤로가기 버튼 추가 (앱처럼 동작)

### 3. 모임 소개 페이지 (`AboutPage.tsx`) - 신규
- 히어로 섹션 (로고 + 슬로건)
- 활동 소개 (언제, 어디서, 뭘 하나요)
- 이런 분께 추천해요
- 수모 안내 (이미지 포함)
- 가입 안내 (회비, 현재 인원, 첫 가입 시 납부 금액)
- 가입 절차 (5단계)
- 자주 묻는 질문 (FAQ)
- CTA 버튼

### 4. FAQ 추가
- "무단 불참하면 어떻게 되나요?" - 1만원 벌금

### 5. 가입 신청 완료 화면 개선
- "홈으로" 버튼 제거, "로그인하기" 버튼만 유지

### 6. 승인 대기(Pending) 회원 화면
- **테스트 계정 추가**: 김대기 (pending@test.com / test123)
- **진행 단계 표시**: 5단계 프로그레스 인디케이터
  1. 가입 신청서 작성 ✓
  2. 가입비 납부 & 승인 대기 (현재 단계)
  3. 카톡방 & 모임통장 초대
  4. 수모 수령
  5. 토요일에 만나요!
- **상태 색상**: 파란색 (대기), 초록색 (활동), 노란색 (휴회)

### 7. 회원 접근 권한 제어
- **Pending 회원 제한**: 회원 명단, 운영 가이드 접근 불가
- **메뉴 순서**: 내 정보 → 회칙 → 회원 명단
- **명칭 변경**: "회원 목록" → "회원 명단"

### 8. 계좌 정보 업데이트
