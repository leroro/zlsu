## 2. 구현된 기능

### 2.1 비로그인 사용자 기능 (3개 페이지)

| 페이지 | 경로 | 기능 |
|--------|------|------|
| 메인 페이지 | `/` | 정원 현황 표시, 바로가기 메뉴 |
| 회칙 페이지 | `/rules` | 회칙 조회 (v1.0, v2.0 선택) |
| 가입 신청 | `/apply` | 신규 회원 가입 신청 폼 |

### 2.2 일반 회원 기능 (4개 페이지)

| 페이지 | 경로 | 기능 |
|--------|------|------|
| 로그인 | `/login` | 이메일/비밀번호 로그인 |
| 내 정보 | `/my` | 내 정보 조회/수정 |
| 회원 목록 | `/members` | 전체 회원 목록 (상태별 필터) |
| 상태 변경 신청 | `/change-status` | 활동/비활동/휴식 변경 신청 |

### 2.3 관리자 기능 (4개 페이지)

| 페이지 | 경로 | 기능 |
|--------|------|------|
| 관리자 홈 | `/admin` | 대시보드, 대기 건수 확인 |
| 가입 신청 관리 | `/admin/applications` | 신청 승인/반려 |
| 회원 관리 | `/admin/members` | 전체 회원 조회, 상태변경 처리 |
| 회원 수정 | `/admin/members/:id` | 개별 회원 정보 수정/삭제 |

### 2.4 공통 기능

- 네비게이션 헤더 (역할별 메뉴 표시)
- 반응형 디자인 (모바일/데스크톱)
- 상태 뱃지 컴포넌트
- 인증/권한 보호 라우팅

```typescript
interface Member {
  id: string;
  email: string;
  password: string;
  name: string;
  nickname?: string;
  phone: string;
  birthDate?: string;
  status: 'active' | 'inactive' | 'resting' | 'withdrawn';
  role: 'admin' | 'member';
  joinedAt: string;
  updatedAt: string;
}
```

```typescript
interface Application {

  phone: string;
  birthDate?: string;
  motivation: string;
  status: 'pending' | 'approved' | 'rejected';
  createdAt: string;
  processedAt?: string;
  processedBy?: string;
```
